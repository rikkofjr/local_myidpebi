# MYIDPEBI

Plugin ini dikhusukan untuk mengelola IDP Karyawan dengan memanfaatkan moodle 4.x keatas

## 🚀 Fitur Utama
* Karyawan
    * Karyawan membuat IDP sesuai dengan kebutuhan pengembangan yang direncakan
    * Karyawan Mengelola aktivitas dari setiap IDP yang dia buat


* Pembimbing
    * Atasan secara default menjadi pembimbing, jika pembimbingnya bukan atasan langsung maka pada field tersebut isi username orang tersebut
    * Pembimbing dapat melakukan Approve terhadap IDP yang telah direncanakan
    * Pembimbing dapat melakukan Verifikasi, jika sudah dilakukan Verifikasi  maka JP akan tampil

## 🛠️ Prasyarat & Instalasi
Sebutkan teknologi utama dan cara menjalankan proyek ini di lokal.
* Prasyarat Moodle
    * Moodle 4.x Keatas

* Setup Custom Profile Field User (ini berfungsi agar atasan dapat diidentifikasi)
    * Tambahkan field shortame = atasan_langsung
    * Field ini berfungsi untuk meletakan username atasan 

* Instalasi
    * Download zip dari repository ini
    * kemudian pada moodle install plugin from zip
